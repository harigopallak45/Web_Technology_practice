Complete CSS Notes

1. Introduction to CSS

CSS (Cascading Style Sheets) is used to design and control the layout of web pages. CSS allows for separation of content and design.

  • Inline CSS: Apply styles directly in HTML elements using the style attribute.
  • Internal CSS: Styles defined inside the <style> tag within the HTML document.
  • External CSS: A separate file with .css extension that is linked to the HTML document using the <link> tag.

2. Inline CSS

Inline CSS is useful when a specific style needs to be applied to a single element.

<p style="color: red; font-size: 20px;">This is a paragraph with inline CSS.</p>

3. Internal CSS

Internal CSS is placed inside the <style> tag in the <head> of an HTML document and applies to the entire document.

<style>
  body {
    background-color: lightgray;
    color: black;
  }
  h1 {
    color: navy;
  }
</style>

4. External CSS

External CSS is written in a separate file and linked to the HTML file using the <link> tag. It provides a way to apply the same style across multiple HTML pages.

<link rel="stylesheet" href="styles.css">

5. CSS Selectors

CSS Selectors are used to select and style HTML elements based on attributes like ID, class, tag name, etc.

  • Tag Selector: Selects all elements with a specific tag name.
    • p { color: blue; }
  • ID Selector: Selects an element with a specific ID, indicated by #.
    • #header { background-color: gray; }
  • Class Selector: Selects all elements with a specific class, indicated by ..
    • .highlight { background-color: yellow; }
  • Group Selector: Applies the same styles to multiple elements.
    • h1, h2, p { color: green; }

6. Pseudo-Classes and Pseudo-Elements

  • Pseudo-Classes: These target elements based on their state.
    • a:hover { color: red; }
  • Pseudo-Elements: Style specific parts of an element, such as the first letter or first line.
    • p::before { content: "Note: "; }

7. CSS Box Model

The box model represents the layout of elements, with properties for margin, border, padding, and content.

  • Content: The content inside the element.
  • Padding: Space around the content, inside the border.
  • Border: A boundary surrounding the padding.
  • Margin: Space outside the border.
div {
  margin: 20px;
  padding: 10px;
  border: 1px solid black;
}

8. Background Properties

CSS background properties are used to apply backgrounds to elements.

  • Background Color: Sets the background color of an element.
    • body { background-color: #f3f3f3; }
  • Background Image: Sets an image as the background of an element.
    • div { background-image: url('image.jpg'); }
  • Background Position: Specifies the position of a background image.
    • div { background-position: top right; }
  • Background Repeat: Specifies how the background image repeats.
    • div { background-repeat: no-repeat; }

9. Borders and Border Radius

  • Border: Specifies the style, width, and color of the element's border.
    • p { border: 2px solid black; }
  • Border Radius: Rounds the corners of an element's border.
    • div { border-radius: 10px; }

10. CSS Gradients

  • Linear Gradient: A gradient that transitions between colors along a straight line.
    • div { background: linear-gradient(to right, red, blue); }
  • Radial Gradient: A gradient that radiates from a central point.
    • div { background: radial-gradient(circle, red, blue); }

11. CSS Flexbox

Flexbox is a layout model that allows for flexible and responsive layouts.

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

12. CSS Grid

CSS Grid Layout allows for two-dimensional grid-based layouts.

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
}

13. CSS Transitions

CSS Transitions allow you to change property values smoothly (over a given duration) rather than having them change instantly.

  • Transition Property: Specifies the CSS property that will undergo a transition.
    • div { transition-property: background-color; }
  • Transition Duration: Specifies how long the transition will take.
    • div { transition-duration: 2s; }
  • Transition Timing Function: Defines the speed curve for the transition (e.g., ease, linear, ease-in, ease-out).
    • div { transition-timing-function: ease-in-out; }
  • Transition Delay: Specifies when the transition will start.
    • div { transition-delay: 0.5s; }

14. CSS Animations

CSS Animations allow for more complex and controlled animations, where elements can transition between multiple styles.

  • @keyframes: Defines the animation sequence.
    • @keyframes example {
  from { background-color: red; }
  to { background-color: yellow; }
}
  • Animation Name: Specifies the keyframe to use for the animation.
    • div { animation-name: example; }
  • Animation Duration: Defines how long the animation lasts.
    • div { animation-duration: 4s; }
  • Animation Iteration Count: Specifies how many times the animation should repeat.
    • div { animation-iteration-count: infinite; }

15. CSS Positioning

CSS positioning properties determine how an element is placed in the layout.

  • Static Position: This is the default positioning method, where elements are placed according to the normal document flow.
    • div { position: static; }
  • Relative Position: An element's position can be adjusted relative to its normal position.
    • div { position: relative; top: 10px; left: 20px; }
  • Absolute Position: Removes the element from the document flow and positions it relative to its nearest positioned ancestor (other than static).
    • div { position: absolute; top: 50px; left: 100px; }
  • Fixed Position: Positions the element relative to the viewport, meaning it stays in place even when the page is scrolled.
    • div { position: fixed; bottom: 10px; right: 10px; }
  • Sticky Position: Combines relative and fixed positioning, where the element is positioned relative until a certain point in the scroll.
    • div { position: sticky; top: 0; }

16. CSS Z-index

The z-index property determines the stacking order of elements. Elements with a higher z-index are displayed in front of those with a lower value.

div { z-index: 10; }

17. CSS Overflow

The overflow property controls what happens when content overflows an element's box.

  • Overflow Visible: The overflow is not clipped, and the content is rendered outside the element's box.
    • div { overflow: visible; }
  • Overflow Hidden: The overflow is clipped, and the rest of the content is hidden.
    • div { overflow: hidden; }
  • Overflow Scroll: The overflow is clipped, but scrollbars appear to allow scrolling.
    • div { overflow: scroll; }
  • Overflow Auto: Similar to scroll, but scrollbars are only added when necessary.
    • div { overflow: auto; }

18. CSS Display

The display property specifies the display behavior of an element.

  • Display Block: The element takes up the full width of its container.
    • div { display: block; }
  • Display Inline: The element takes up only as much space as its content, and does not start on a new line.
    • span { display: inline; }
  • Display Inline-Block: Similar to inline, but allows for setting width and height.
    • img { display: inline-block; }
  • Display None: Hides the element; it is removed from the document flow entirely.
    • div { display: none; }

19. CSS Visibility

The visibility property controls the visibility of an element, but unlike display: none;, it keeps the space allocated for the element in the layout.

  • Visibility Visible: The element is visible (default).
    • div { visibility: visible; }
  • Visibility Hidden: The element is hidden, but the space it occupies remains.
    • div { visibility: hidden; }

20. CSS Units

CSS supports a variety of units for length, including both absolute and relative units.

  • Absolute Units: These units are fixed and do not change with the size of the viewport.
    • px - Pixels
    • pt - Points (1pt = 1/72 of an inch)
    • cm - Centimeters
  • Relative Units: These units change based on the size of other elements, such as the font size of the parent.
    • % - Percentage relative to the parent element.
    • em - Relative to the font size of the element.
    • rem - Relative to the font size of the root element.
    • vw - Viewport width (1vw = 1% of the viewport's width).
    • vh - Viewport height (1vh = 1% of the viewport's height).

21. CSS Media Queries

Media queries allow you to apply CSS rules based on the characteristics of the user's device, such as the width of the screen.

@media (max-width: 600px) {
  div {
    background-color: lightblue;
  }
}

Media queries are useful for creating responsive designs that adjust based on the screen size.

19. CSS Visibility (continued)

Visibility also includes:

  • Visibility Collapse: Used with table rows or columns to collapse them while maintaining their layout space.
    • tr { visibility: collapse; }

20. CSS Units (continued)

Additional units include:

  • ch: Relative to the width of the "0" (zero) character in the font.
    • div { width: 10ch; }
  • ex: Relative to the height of the lowercase "x" character in the font.
    • div { height: 5ex; }

22. CSS Flexbox

Flexbox is a layout model that allows you to design complex layouts easily and predictably.

  • Display Flex: Enables flex container on an element.
    • container { display: flex; }
  • Flex Direction: Defines the direction of the flex items (row, column).
    • container { flex-direction: row; }
  • Justify Content: Aligns flex items along the main axis.
    • container { justify-content: center; }
  • Align Items: Aligns flex items along the cross axis.
    • container { align-items: stretch; }
  • Flex Wrap: Specifies whether flex items should wrap onto multiple lines.
    • container { flex-wrap: wrap; }

23. CSS Grid

CSS Grid Layout provides a two-dimensional grid-based layout system.

  • Display Grid: Turns an element into a grid container.
    • container { display: grid; }
  • Grid Template Columns: Defines the number and size of columns in the grid.
    • container { grid-template-columns: 1fr 2fr; }
  • Grid Template Rows: Defines the number and size of rows in the grid.
    • container { grid-template-rows: 100px 200px; }
  • Grid Area: Defines a grid itemâ€™s area within the grid.
    • item { grid-area: header; }
  • Gap: Defines the space between grid items.
    • container { gap: 10px; }

24. CSS Variables

CSS Variables (Custom Properties) allow you to store values for reuse throughout your CSS.

  • Define Variable: Variables are defined with a -- prefix.
    • :root {
  --main-color: coral;
}
  • Use Variable: Use the var() function to apply the variable.
    • div {
  background-color: var(--main-color);
}

25. CSS Pseudo-classes

Pseudo-classes apply styles to elements based on their state or position in the document.

  • :hover: Applies styles when the user hovers over an element.
    • a:hover { color: red; }
  • :focus: Applies styles when an element is focused.
    • input:focus { border-color: blue; }
  • :nth-child(n): Applies styles to the nth child of a parent.
    • li:nth-child(2) { color: green; }
  • :first-child: Applies styles to the first child of a parent.
    • p:first-child { font-weight: bold; }
  • :last-child: Applies styles to the last child of a parent.
    • p:last-child { font-style: italic; }

26. CSS Pseudo-elements

Pseudo-elements create virtual elements that can be styled.

  • ::before: Inserts content before an element's content.
    • p::before {
  content: "Note: ";
  font-weight: bold;
}
  • ::after: Inserts content after an element's content.
    • p::after {
  content: " (End of paragraph)";
  font-style: italic;
}

27. CSS Content Property

The content property is used with pseudo-elements to generate content.

  • Using Content Property: Specifies the content to insert with ::before or ::after.
    • p::before {
  content: "Note: ";
}
  • Content Values: Can include text, images, and counters.
    • p::after {
  content: url('icon.png');
}

28. CSS Gradients

CSS Gradients are used to create smooth transitions between colors.

  • Linear Gradient: Creates a gradient along a straight line.
    • background: linear-gradient(to right, red, yellow);
  • Radial Gradient: Creates a gradient emanating from a center point.
    • background: radial-gradient(circle, red, yellow);

29. CSS Filter

The filter property applies graphical effects like blur or color adjustments to elements.

  • Blur: Applies a blur effect.
    • img { filter: blur(5px); }
  • Brightness: Adjusts the brightness of an image.
    • img { filter: brightness(150%); }
  • Contrast: Adjusts the contrast of an image.
    • img { filter: contrast(200%); }

30. CSS Transformations

The transform property allows you to apply transformations to elements, such as translation, rotation, scaling, and skewing.

  • Translate: Moves an element from its current position.
    • transform: translate(50px, 100px);
  • Rotate: Rotates an element around its origin.
    • transform: rotate(45deg);
  • Scale: Scales an element up or down.
    • transform: scale(1.5);
  • Skew: Skews an element along the X or Y axis.
    • transform: skew(30deg, 20deg);

31. CSS Transitions

The transition property allows you to change property values smoothly over a specified duration.

  • Transition Property: Specifies the CSS properties to transition.
    • transition: all 0.5s ease;
  • Transition Duration: Defines the duration of the transition.
    • transition-duration: 1s;
  • Transition Timing Function: Defines the speed curve of the transition.
    • transition-timing-function: ease-in-out;
  • Transition Delay: Sets a delay before the transition starts.
    • transition-delay: 0.2s;

32. CSS Animations

CSS Animations allow you to create complex animations by defining keyframes.

  • @keyframes: Defines the animation sequence.
    • @keyframes slide {
  from { transform: translateX(-100%); }
  to { transform: translateX(0); }
}
  • Animation Property: Applies the animation to an element.
    • animation: slide 2s ease-in-out;
  • Animation Duration: Defines how long the animation lasts.
    • animation-duration: 2s;
  • Animation Timing Function: Specifies the timing function of the animation.
    • animation-timing-function: ease-in-out;
  • Animation Delay: Sets a delay before the animation starts.
    • animation-delay: 0.5s;
  • Animation Iteration Count: Defines how many times the animation should repeat.
    • animation-iteration-count: infinite;

33. CSS Overflow

The overflow property controls the content overflow of an element.

  • Visible: Content is not clipped and may overflow.
    • overflow: visible;
  • Hidden: Content is clipped and not visible.
    • overflow: hidden;
  • Scroll: Content is clipped, but scrollbars are added.
    • overflow: scroll;
  • Auto: Scrollbars are added only when necessary.
    • overflow: auto;

34. CSS Object Fit

The object-fit property specifies how the content of an element should be resized to fit its container.

  • Contain: Scales the content to fit the container while maintaining its aspect ratio.
    • img { object-fit: contain; }
  • Cover: Scales the content to cover the container while maintaining its aspect ratio.
    • img { object-fit: cover; }
  • Fill: Stretches the content to fill the container.
    • img { object-fit: fill; }
  • None: Does not resize the content.
    • img { object-fit: none; }
  • Scale Down: Scales the content to the smaller of the two options between contain and cover.
    • img { object-fit: scale-down; }

35. CSS Clip Path

The clip-path property allows you to create a clipping region to show a part of an element.

  • Basic Shapes: Creates clipping paths using basic shapes.
    • clip-path: circle(50%);
  • Polygon: Defines a custom clipping path using a polygon.
    • clip-path: polygon(50% 0%, 100% 100%, 0% 100%);
  • Inset: Clips an element using an inset rectangle.
    • clip-path: inset(10% 20% 30% 40%);

36. CSS Z-Index

The z-index property specifies the stack order of elements. Higher values are in front of lower values.

  • Stacking Context: Defines the stacking order of positioned elements.
    • div { position: absolute; z-index: 10; }
  • Auto: Default value, the element inherits the z-index value from its parent.
    • div { position: absolute; z-index: auto; }

37. CSS Alignment

Alignment properties in CSS help position elements within containers.

  • Text Align: Aligns text within its container.
    • p { text-align: center; }
  • Vertical Align: Aligns inline or table-cell elements vertically.
    • img { vertical-align: middle; }
  • Align Self: Aligns a flex item inside its flex container.
    • .item { align-self: flex-start; }
  • Align Content: Aligns flex container lines within the flex container.
    • .container { align-content: space-between; }

38. CSS Overflow Wrap

The overflow-wrap property specifies whether the browser can break words and wrap text.

  • Normal: Allows words to break normally.
    • p { overflow-wrap: normal; }
  • Break-word: Allows long words to be broken and wrap onto the next line.
    • p { overflow-wrap: break-word; }

39. CSS Writing Modes

The writing-mode property specifies the direction in which text is written.

  • Horizontal-TB: Text flows horizontally from left to right.
    • body { writing-mode: horizontal-tb; }
  • Vertical-LR: Text flows vertically from top to bottom.
    • body { writing-mode: vertical-lr; }
  • Vertical-RL: Text flows vertically from bottom to top.
    • body { writing-mode: vertical-rl; }
  • Sideways: Allows sideways text.
    • p { writing-mode: sideways-lr; }

40. CSS Logical Properties

Logical properties provide a way to specify layout and spacing relative to the writing mode of the document.

  • Margin Inline: Sets the inline margin.
    • div { margin-inline: 20px; }
  • Padding Block: Sets the block padding.
    • div { padding-block: 10px; }
  • Border Block Start: Sets the border on the start side of the block.
    • div { border-block-start: 2px solid black; }

41. CSS Grid Layout

The grid property allows you to create complex layouts using rows and columns.

  • Grid Container: Defines the container for grid items.
    • .container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: repeat(2, 100px);
}
  • Grid Items: Items placed within the grid container.
    • .item {
  grid-column: span 2;
  grid-row: 1;
}
  • Grid Template Columns: Defines the number and size of columns.
    • grid-template-columns: 1fr 2fr 1fr;
  • Grid Template Rows: Defines the number and size of rows.
    • grid-template-rows: 100px 200px;
  • Grid Area: Allows you to define an area within the grid.
    • .item {
  grid-area: header;
}

42. CSS Flexbox

The flex property provides a way to distribute space and align items within a container.

  • Flex Container: Defines the container for flex items.
    • .container {
  display: flex;
  flex-direction: row;
  justify-content: center;
  align-items: stretch;
}
  • Flex Direction: Specifies the direction flex items are placed in the flex container.
    • flex-direction: row; /* or column */
  • Justify Content: Aligns items horizontally within the container.
    • justify-content: center; /* or flex-start, flex-end, space-between, space-around */
  • Align Items: Aligns items vertically within the container.
    • align-items: center; /* or flex-start, flex-end, stretch */
  • Flex Grow: Defines the ability for a flex item to grow if necessary.
    • .item {
  flex-grow: 1;
}
  • Flex Shrink: Defines the ability for a flex item to shrink if necessary.
    • .item {
  flex-shrink: 1;
}
  • Flex Basis: Defines the default size of a flex item before space distribution.
    • .item {
  flex-basis: 100px;
}

43. CSS Variables

CSS variables (custom properties) allow you to store values that can be reused throughout a document.

  • Define Variables: Variables are defined within a selector using a custom property name.
    • :root {
  --main-color: #3498db;
  --padding: 10px;
}
  • Use Variables: Variables are used by referencing their names.
    • body {
  color: var(--main-color);
  padding: var(--padding);
}

44. CSS Filter Effects

The filter property applies graphical effects like blurring or color shifting to elements.

  • Blur: Applies a blur effect to an element.
    • img { filter: blur(5px); }
  • Brightness: Adjusts the brightness of an element.
    • img { filter: brightness(0.8); }
  • Contrast: Adjusts the contrast of an element.
    • img { filter: contrast(1.5); }
  • Saturate: Adjusts the saturation of an element.
    • img { filter: saturate(2); }
  • Grayscale: Converts the element to grayscale.
    • img { filter: grayscale(100%); }
  • Sepia: Applies a sepia effect to an element.
    • img { filter: sepia(100%); }
  • Hue Rotate: Adjusts the hue of an element.
    • img { filter: hue-rotate(90deg); }

45. CSS Logical Properties

Logical properties provide a way to specify layout properties relative to the writing mode of the document, enabling better support for different writing modes and directions.

  • Margin Inline Start: Sets the margin on the inline start side.
    • .element { margin-inline-start: 20px; }
  • Padding Block End: Sets the padding on the block end side.
    • .element { padding-block-end: 15px; }
  • Border Inline End: Sets the border on the inline end side.
    • .element { border-inline-end: 2px solid black; }
  • Border Block Start: Sets the border on the block start side.
    • .element { border-block-start: 2px solid black; }

46. CSS Multi-Column Layout

The multi-column layout allows you to create multiple columns for text.

  • Column Count: Defines the number of columns.
    • .column-container {
  column-count: 3;
}
  • Column Width: Specifies the width of columns.
    • .column-container {
  column-width: 200px;
}
  • Column Gap: Sets the space between columns.
    • .column-container {
  column-gap: 20px;
}

47. CSS Grid Auto Flow

The grid-auto-flow property controls how the auto-placement algorithm works in CSS Grid Layout.

  • Row: Items are placed into rows by default.
    • .container { grid-auto-flow: row; }
  • Column: Items are placed into columns.
    • .container { grid-auto-flow: column; }
  • Dense: Fills in gaps with items if possible.
    • .container { grid-auto-flow: dense; }

48. CSS Flexbox Alignment

The flexbox alignment properties help to align flex items and containers.

  • Align Content: Aligns flex lines within the flex container.
    • .container { align-content: space-between; }
  • Align Items: Aligns items along the cross axis.
    • .container { align-items: center; }
  • Align Self: Allows a flex item to override the align-items property.
    • .item { align-self: flex-end; }

49. CSS Grid Template Areas

The grid-template-areas property allows you to name areas of a grid and place items into those areas.

  • Define Areas: Create named areas in the grid.
    • .container {
  grid-template-areas:
    "header header header"
    "main main sidebar"
    "footer footer footer";
}
  • Assign Areas: Assign items to named grid areas.
    • .header { grid-area: header; }
.main { grid-area: main; }
.sidebar { grid-area: sidebar; }
.footer { grid-area: footer; }

50. CSS Custom Properties

Custom properties (CSS variables) can be used to store and reuse values in CSS.

  • Define Variables: Variables are defined with a -- prefix.
    • :root {
  --main-bg-color: lightblue;
  --main-text-color: darkblue;
}
  • Use Variables: Variables are used with the var() function.
    • body {
  background-color: var(--main-bg-color);
  color: var(--main-text-color);
}
